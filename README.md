# Weather tool:
* Prompt: Use this tool to get weather updates
* Endpoint: `https://grateful-lemming-clearly.ngrok-free.app/weather`
* Sample JSON input"
```json
{
  "city": "Name of the city to get the weather"
}
```
* Sample JSON output:
```json
{
  "city": "City name",
  "country": "Country name",
  "temperature": "Temperature",
  "feels_like": "Feels like",
  "humidity": "Humidity",
  "pressure": "Pressure",
  "weather": "Weather",
  "wind_speed": "Wind speed",
  "clouds": "Clouds"
}
```
